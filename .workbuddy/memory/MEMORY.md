# 1000天产品 - 长期记忆

## 产品概述
**每日英语听说1000天-完整版** - 一个包含641天英语学习内容的完整版产品，部署在GitHub Pages上。

## 产品定位
- **小名**：1000天产品
- **定位**：完整版，定版后不再修改
- **内容**：641天预生成内容（2026年276天 + 2027年365天）
- **特点**：永不过时，日期显示为"第N天"格式

## 访问地址
- GitHub Pages：https://lnfkstz20260326.github.io/daily-words-1000/
- GitHub仓库：https://github.com/lnfkstz20260326/daily-words-1000

## 技术架构
- **文件**：641个HTML文件 + index.html
- **学习进度**：localStorage存储
- **颜色状态**：
  - 红色：未学习
  - 蓝色：当前进度
  - 绿色：已学习

## 朗读规则（与当天产品相同）
**5个按钮的功能：**
1. 🐢 慢速词×3：40%速度读3遍英语 + 1遍汉语
2. 🚀 匀速词×3：80%速度读3遍英语（无汉语）
3. 🐢 慢速句×3：60%速度读3遍英语 + 1遍汉语
4. 🚀 匀速句×3：80%速度读3遍英语（无汉语）
5. ⚡ 常速句×3：90%速度读3遍英语（无汉语）

## 与当天产品的区别
| 项目 | 当天产品 | 1000天产品 |
|------|----------|------------|
| 生成方式 | 每天自动生成 | 预生成641天 |
| 日期显示 | YYYY-MM-DD | 第N天 |
| 学习进度 | 无追踪 | localStorage追踪 |
